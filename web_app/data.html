Data Used for Performing The Analysis

This page describes how the data to perform the analysis has been extracted. In order to easily download tweets, a Twitter Developer account is needed. Access to such an account can be applied for at Apply for access. With this account, the Twitter API can be utilized. The Twitter API only get one access to to exctract the most reason 3200 tweets from a given account (3200 tweet do not go far back for many american polticians). However, Twitter's Terms of Service do allow for datasets of tweets ID's to be distributed to third parties (not the full JSON). Tweets dating back can be extracted if one has the ID's of the tweets.

We found two sources that containing open tweet-id's related to our project, and another source that stored the full length of the tweets made by Donald Trump. The three sources are listed below:

115th U.S. Congress Tweet Ids
A open dataset with 2,041,399 tweets from the Twitter accounts of members of the 115th U.S. Congress collected in the period of January 27, 2017 and January 2, 2019. The dataset consists of two files of interest - namely
  • senators-1.txt: that contains tweet ids for Seneators
  • representatives-1.txt: that contains tweet ids for Representatives
Littman, Justin, 2017, "115th U.S. Congress Tweet Ids, https://doi.org/10.7910/DVN/UIVHQR, Harvard Dataverse, V5.

116th U.S. Congress Tweet Ids
A open dataset with 2,817,747 tweets from the Twitter accounts of members of the 116th U.S. Congress collected in the period of January 27, 2019 and May 7, 2020. The dataset consists of two files of interest - namely
  • Senators: congress116-senate-ids.txt` that contains tweet ids for Seneatorss
  • Representatives: congress116-house-ids.txt` that contains tweet ids for Representatives
Wrubel, Laura; Kerchner, Daniel, 2020, "116th U.S. Congress Tweet Ids, https://doi.org/10.7910/DVN/MBOJNS, Harvard Dataverse, V5.

Trump Twitter Archive
A site dedicated to scrape every single tweet from Donald J. Trump. Here we downloaded all tweets in the periods of January 27, 2017 and January 2, 2019 and January 27, 2019 and May 7, 2020. Trump Twitter Archive
