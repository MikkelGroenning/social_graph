'Who Follows Whom'-Graph Analysis

In this section results from the follow-graph will be presented. The follow-graph is constructed based on which profiles that follow each other on Twitter. In other words if e.g. a Senator follows another Senator on Twitter a directed edge between them is constructed. The idea is illustrated in the figure below. In this graph we consider the three politicans Dean Heller, Alexandria Ocasio-Cortez and Bernie Sanders as nodes. There is a directed edge between Dean Heller and Alexandria Ocasio-Cortez which indicates that Dean Heller follows Alexandria Ocasio-Cortez on Twitter. As Alexandria Ocasio-Cortez does not follow Dean Heller there is not edge going the opposite direction. Following the analogy it can be seen that Bernie Sanders follows Alexandria Ocasio-Cortez and vice versa. Dean Heller follows Bernie Sander but Bernie Sanders does not follow Dean Heller. This is the type of graph that will be created for all 618 politicans in our dataset.

Constructing the Graph

The graph is constructed by looking up every the Twitter profile of each politican using the Twitter API. Here it is extracted who the profile follows and directed links are created between the politicans as described above. The total progress can be seen in Get_follow_Adjancency notebook or here.

The created graph contains 618 nodes but as three politicians act as isolates, they are removed. Thus, the graph considered contains 615 nodes with 83778 edges. With that many edges the network can definitely be considered as dense since each politician follows approximately 137 of the other politicians, on average. This corresponds to around 22% of all polticians. Following this info it will be investigated if patterns can be seen in who follows whom.

The Follow-Graph

Below the comple follow-graph can be seen. Note that the edges have only been drawn if there exists a link going in both directions between a node. This is done since the edges would otherwise become to overwhelming. Furthermore, only the Giant Connected Component is displayed. The interactive window allows for removing edges and nodes by clicking the legend to the right and hovering over the nodes revieval who the politicians are.

It can be seen that the network is super-dense! In fact the average shortage path lengh is just 1.81. The number can be confirmed by examining the explainer-notebook. It only takes around two steps on average to get to another node from any node in the graph.

The graph is so dense it contains a clique of 51 nodes - all Democrats. The second largest clique is of 33 nodes consisting of only Republicans.

Assortativity

In the initial analysis of the edges, it was found how politicians are more likely to follow others from their own party compared to other parties. In this section the assortativity across Parties, Type and State is determined. The results are summarized in the table below.

Assortativity
Party 0.812
Type 0.576
State 0.065

The assortativity of nodes from same party is 0.81 - a very high measure. But this is not surprising given the structure of the graph. It clearly tells that politicians from the same parties like to link up.

From the plot of the graph it also looks like the senators are separated from the representatives. This can be confirmed from the assortativity of 0.58. Eventhough it is lower than assortativity across parties it is still a clear positive assortativity meaning that the politicians like to link up to other politicians of the same type.

Across polticians from the same state a much lower assortativity value is found to be 0.065, so still positive. Off course there are also way more states, where many of them only have a few politician which off course makes it harder to find very high assortativity.

Community Detection

The last analysis that will be carried out on the graph is community detection. First communities based on the data about the politicians will be investigated before looking at the Louvain method to create communities based on the actual links in the Who-follows-whom graph.

Communities Based On Party

The first natural split that will be investigated is based on parties. Recall that all but three node were either Democrat or Republican - namely the two independent politicians Bernie Sanders and Angus King as well as the libertarian politician Justin Amash. Here it will be explored which of the two large parties each politician is more likely to support. The approach to this analysis will be to see how these three politicians should be placed to obtain the highest modularity. Now, for the three last politicians there are (\2^3=8\) combinations. In the ideal split the modularity based on the party split is 0.400.

Communities Based On States

The next split is based on states. The modularity obtained is 0.063 which is much worse that the party-split. Now there are 56 communities where some are very small which is why it make sense to try to increase it by merging the states.

Trying To Merge States

The states are merged using the Louvain algorithm. This is done by first making a new weighted graph with 56 nodes - one corresponding to each state. The edges are now weighted where the weight between two states are given as the total number of links between politicians from these two states. This forces the first iteration in the Louvain method to first merge on a state level. Now treating the states as new nodes the algorithm can be used to find the best state partition. The resulting communities across states are shown on the interactive map below.

The overseas territories are not included in the above plot. Their communities can be read in list below:

This is a very interesting map if one follows US politics. Community 1 seems to include many of usual Republican states through the central and South-East America. Community 1 on the other hand is primarily Democratic states in New England and at the west coast. Community 2 is also some usual Democratic states. Community 3-7 contains only one state and it is states that is usually more divided and considered swing states in the presidential election (Texas is still primarily Republican). It is interesting that they are their own community here. Lastly, community 7 is a bit harder to explain.

The modularity of the state communities are 0.134 higher than only considering one state but it is still much worse that the party split.

Communities Based On The Louvain Algorithm

The best partition for communities are extracted using the Louvian algorithm. Upon convergence the algorithm find a total of 3 communities with modularity of 0.448. It is not that much better than the obtained values in the party split which is an indication that the party split wasn't actually that bad. Subconclusion

In this part, a graph based on how politicians follow each other has been created, visualized, and analyzed. The graph was very dense! Overall it was found that representatives very clearly followed others from their own party while the senators were more likely to follow other senators. A lot of different tools for analyzing networks were used and the conclusion can be summarized to the fact that politicians are by a clear margin more likely to follow others with the same beliefs as themselves - this includes a split between senators and representatives too. When using the Louvain method three communities were found, that can be summarized as senators, Republican representatives, and Democratic representatives.
